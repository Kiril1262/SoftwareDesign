# 🍬 Sweet Blog — навчальний PHP-проєкт

## Опис
Sweet Blog — це демонстраційний проєкт, який ілюструє застосування шаблонів проєктування в PHP. Кожен файл репрезентує окремий патерн.

## 📦 Структура

```
sweet_blog/
├── index.php
├── core/
│   ├── Element.php
│   ├── LifecycleTemplate.php
│   ├── Iterator.php
│   ├── Command.php
│   ├── State.php
│   └── Visitor.php
│── task/
```├── style.css

## 🔧 Використані шаблони проєктування

- **Шаблонний метод**  
  *Файл: `core/LifecycleTemplate.php`*  
  Абстрактний клас, який задає каркас для рендерингу елемента.

- **Основний елемент**  
  *Файл: `core/Element.php`*  
  Наслідує шаблонний метод та додає логіку відображення HTML.

- **Ітератор**  
  *Файл: `core/Iterator.php`*  
  Реалізує обхід DOM-подібної структури елементів.

- **Команда**  
  *Файл: `core/Command.php`*  
  Додає стилі до елемента через окремий об’єкт-команду.

- **Стейт**  
  *Файл: `core/State.php`*  
  Перемикає статус блогу між «Чернетка» та «Опубліковано».

- **Відвідувач**  
  *Файл: `core/Visitor.php`*  
  Рекурсивно рахує кількість HTML-елементів у дереві.

- **Стилі**  
  *Файл: `task/style/style.css`*  
  Рекурсивно рахує кількість HTML-елементів у дереві.

## ▶️ Запуск
Просто відкрийте `index.php` у браузері або запустіть локальний сервер:

```bash
php -S localhost:8000
```

## 🔗 Головний файл
*Файл: `index.php`*  
Містить приклади використання всіх шаблонів у єдиному рендері.
